Over de taal

Het omzetten van de natuurlijke taal naar door de computer uitvoerbare code heeft drie stappen:

  1. In deze stap specificeert de regelmaker zijn regels in de gecontroleerde natuurlijke taal
  2. De gecontroleerde gestructureerde taal wordt omgezet naar een tussentaal die speciaal ontworpen is voor regelgebaseerd werken. We hebben gekozen voor de Drools Rule Language (DRL)
  3. We gebruiken de KIE execution server van Drools om de DRL op de computer uitvoerbaar te maken op de Java Virtual Machine (JVM).

Het ecosysteem

Act native is slechts een bouwblok binnen een ecosysteem. Simpelgezegd kan de Act Native bouwsteen op basis van een set aan regels voor een specifiek geval een beslissing nemen. De bouwsteen accepteert een set en regels en een instantie (de data waarop de beslissing genomen moet worden).

Open source

Act Native is gebouwd op basis van open source componenten. Dit betekent overigens niet dat de broncode en de data in de systemen open zijn.
